3.6. Interrupt status register

HW prefix: cbmia_eic_isr
HW address: 0xb
C prefix: EIC_ISR
C offset: 0x2c

Each bit represents the state of corresponding interrupt. 1 means the interrupt is pending. Writing 1 to a bit clears the corresponding interrupt. Writing 0 has no effect.

  • RX_DONE [read/write]: mil1553 frame received.
    read 1: interrupt 'mil1553 frame received.' is pending
    read 0: interrupt not pending
    write 1: clear interrupt 'mil1553 frame received.'
    write 0: no effect
